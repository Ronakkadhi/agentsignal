API

GET /feed Ranked markdown feed
GET /feed?format=json Structured JSON
GET /feed?topic=ai&min_score=70 Filtered high-signal only
GET /feed?after=<timestamp> Delta updates (only new)
POST /ask Ask a question, get signal-grounded answer
POST /watch Create topic subscription
GET /feed?watch=<id> Personalized watch feed
GET /sources List active sources

Ask Anything

Push a question, get an answer grounded in real-time signals.

Terminal 
curl -X POST agentsignal.co/ask \
  -H "Content-Type: application/json" \
  -d '{"q": "What happened to crypto today?"}'

Watch Topics

Subscribe to topics. Your agent gets a personalized, filtered feed.

Terminal 
curl -X POST agentsignal.co/watch \
  -d '{"topics": ["ai", "regulation"], "min_score": 60}'

# → {"watch_id": "w_abc123", "feed_url": "/feed?watch=w_abc123"}

curl agentsignal.co/feed?watch=w_abc123
