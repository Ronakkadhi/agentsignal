Three Ways to Use

Pull signals. Ask questions. Follow topics.

Pull Get ranked signals 
curl agentsignal.co/feed?topic=ai&min_score=70
Ask Push a question, get a signal-grounded answer 
curl -X POST agentsignal.co/ask \
  -d '{"q": "What happened to crypto today?"}'
Follow Subscribe to topics, get a personalized feed 
curl -X POST agentsignal.co/follow \
  -d '{"topics": ["ai", "regulation"], "min_score": 60}'

# Then poll your feed:
curl agentsignal.co/feed?follow=f_abc123
